FAQ / よくある質問

よくある質問

GPU がない PC でも使えますか?

使えます。GPU なしの場合は CPU で処理し、60 分の会議で約 1 時間が目安です(処理中は放置で OK)。NVIDIA GPU 搭載 PC なら同じ会議が約 3 分で完了します。アプリが自動で判定するので設定は不要です。
※ 高速処理に対応するのは NVIDIA GPU(CUDA)のみです。Intel / AMD の内蔵グラフィックスは高速処理の対象外で、CPU 処理となります(設定項目もありません)。

本当にインターネットへ送信しないのですか?

音声・文字起こし・議事録の内容を外部へ送信することは一切ありません。ネットワークを使うのは、初回起動時の AI モデルのダウンロードと手動の更新確認のみです。オフライン環境でも(モデル取得後は)動作します。

どんなファイル形式に対応していますか?

MP3 / WAV / M4A / MP4 など、一般的な音声・動画ファイルに対応しています。Teams や Zoom の録画ファイルもそのまま放り込めます。

動作環境を教えてください

Windows 10 / 11(64bit)、メモリ 8GB 以上を推奨。NVIDIA GPU(VRAM 6GB 以上)があると高速モードで動作します。AI モデルの保存に約 5GB のディスク空き容量が必要です。
